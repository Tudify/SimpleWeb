## controls
### Smart Bar
⌘ + U / ctrl + U
### New tab
⌘ + T / ctrl + T
### Close Tab
⌘ + W / ctrl + W
### Refresh page
⌘ + R / ctrl + R
### Settings page
⌘ + U / ctrl + U, then type tudify://settings.
